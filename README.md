# ✈️ AI Flight Delay Prediction System

## Project Overview

The AI Flight Delay Prediction System is a Machine Learning project that predicts whether a flight will be delayed or arrive on time.

The model learns from historical flight information and identifies patterns affecting flight delays.

---

# Objective

To develop a predictive machine learning model capable of forecasting flight delays based on flight parameters.

---

# Features

- Data preprocessing
- Exploratory Data Analysis
- Machine Learning model training
- Flight delay prediction
- Model accuracy evaluation
- Visualization of flight delay patterns

---

# Technologies Used

## Programming Language
- Python

## Libraries

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

# Machine Learning Algorithms

The following algorithms were implemented:

## 1. Logistic Regression

Used as a baseline classification model.

## 2. Random Forest Classifier

Used as the main prediction model due to better performance and ability to handle complex patterns.

---

# Dataset Features

| Feature | Description |
|-|-|
| Airline | Airline name |
| Distance | Flight distance |
| Departure Hour | Scheduled departure time |
| Weather | Weather condition |
| Previous Delay | Previous delay status |
| Delayed | Target variable |
